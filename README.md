# TicketFlow

TicketFlow is a modern, responsive ticketing system built with React and Vite.

## Features

- **Dashboard**: Real-time overview of ticket stats (Total, Pending, Resolved, Critical).
- **Ticket Management**: Create, edit, and delete tickets.
- **Filtering & Sorting**: Filter by status and sort by various fields.
- **Settings**: Dynamic management of dropdown options (Status, Priority, Category).
- **Persistence**: Data is persisted locally using `localStorage`.
- **Responsive Design**: Optimized for various screen sizes.

## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.

## Technologies

- React
- Vite
- React Router
- Lucide React (Icons)
- CSS Modules / Vanilla CSS
